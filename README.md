## Installating Dev Environment

```
composer install
npm install

php artisan migrate
php artisan db:seed
php artisan monitor:check-uptime

cd lighthouse/
npm install -g lighthouse
npm install
node init

cd ../
php artisan serve
npm run prod - in another terminal
```
